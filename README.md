# AWS Django Task Manager

A Django-based Task Management web application deployed on AWS.

## Live Project
http://52.66.114.96

## Features
- View task list
- Task descriptions
- Django admin panel
- Static files managed with AWS S3
- Deployed on AWS EC2

## Tech Stack
- Python
- Django
- AWS EC2
- AWS S3
- AWS IAM
- Gunicorn
- Nginx
- Git & GitHub
- SQLite

## Project Description
This project is a cloud-deployed task management application built using Django.  
It was deployed on an AWS EC2 instance using Gunicorn and Nginx, with static files uploaded to AWS S3.
